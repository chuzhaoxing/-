##区块
区块引用则使用 email 形式的 '>' 角括号,如下

>

##修辞和强调
使用星号和底线来标记需要强调的区段，如下
>*星号* _下划线_  
>**星号**  __划线__  
>***星号***

    public static function substr_cut($str, $lenght)
    {
        $tmpstr = "";
        $strlen = strlen($str);
        $tmpLen = 0;
        for($i = 0; $i < $strlen; $i++){
            $tmpLen++;
            if ($tmpLen>$lenght) {
                return $tmpstr;
            }
            if(ord(substr($str, $i, 1)) > 127){
                $tmpstr.=substr($str, $i, 3);
                $i+=2;
            }else
                $tmpstr.= substr($str, $i, 1);
        }
        return $tmpstr;
    }
